# Open Source Contributions


### [RabitQ-Library](https://github.com/VectorDB-NTU/RaBitQ-Library)
RabitQ-Library is a C++ library for high performance vector search and retrieval using the quantization technique RabitQ.

- Added full AVX2 support for SIMD operations
- Restructured memory layout for improved operations efficiency with 15% performance boost
- Fixed incorrect SIMD intrinsics usage for AVX512
- Introduced testing framework and unit tests

### [MRT-Buddy](https://github.com/aniruddha-adhikary/mrt-buddy)
MRT Buddy is an unofficial community-driven Android app designed to check the balance of Dhaka MRT Card. The app provides a convenient way for 100k+ users to monitor their card balance and transaction history.

- Reverse engineered the binary data format used by the MRT card to extract timestamp data

### [spring-tx-board](https://github.com/Mamun-Al-Babu-Shikder/spring-tx-board)

Spring Tx Board is an intelligent, auto-configurable transaction monitoring and diagnostics tool for Spring-based applications.

- Integrated with Spring Boot Actuator to collect transaction metrics and health information

### [Relevant-Leetcode](https://github.com/nikhilm25/RelevantLeetcode)

A website that shows the most frequently asked questions by companies.

- Added feature to import and export solve startus of problems to sync between multiple devices

### [Apache Nutch](https://github.com/apache/nutch)
Nutch is a highly extensible, highly scalable, matured, production-ready Web crawler which enables fine grained configuration and accomodates a wide variety of data acquisition tasks.

- Extended the Metadataclass to support case insensitive properties in compliance with the HTTP specification

### [Apache StormCrawler](https://github.com/apache/stormcrawler)

Apache StormCrawler is an open source collection of resources for building low-latency, scalable web crawlers on Apache Storm. It is provided under Apache License and is written mostly in Java.


-Replaced the deprecated URL constructor with the recommended URI class to improve code reliability and maintainability
- Refactored the codebase to remove magic numbers and replace them with named constants, improving code readability and maintainability

### [Apache kie-drools](https://github.com/apache/incubator-kie-drools)

Drools is a rule engine, DMN engine and complex event processing (CEP) engine for Java

- Clean up of the codebase by removing unused imports and variables, improving readability and maintainability
